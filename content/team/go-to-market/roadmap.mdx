# GTM Roadmap

GTM-facing delivery sequencing and narrative timing. Maps engineering milestones to public-facing messages.

## The two-layer GTM funnel

Synap's distribution follows a deliberate funnel architecture:

```
Layer 0: OpenClaw ecosystem (free, open-source, ClawHub skills)
  │  → 250K+ potential users, zero cost acquisition
  │  → Entry point: synap-memory skill on ClawHub
  │  → Distribution: Discord, Reddit, HN, content creators
  ▼
Layer 1: Managed pod ($15-20/mo subscription)
  │  → Revenue engine, recurring revenue
  │  → Value prop: sovereignty + AI + no maintenance
  │  → Upsell path from Layer 0 self-hosted users
  ▼
Layer 2: Visual products ($50/mo)
  │  → Upsell: advanced views, AI workflows, team collaboration
  │  → Enterprise tier: multi-workspace, SSO, audit trails
  │  → Maximum ARPU from power users and teams
```
