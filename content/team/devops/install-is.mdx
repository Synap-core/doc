# Install Intelligence Service

First-time IS setup provisions orchestration, networking, and observability dependencies.

- Run the scripted install flow (`./synap install`) from the service deploy surface.
- Configure Docker runtime, Traefik ingress, and tracing/analytics connectors.
- Validate Control Plane connectivity before enabling production traffic.

Follow with [Monitoring Setup](/team/devops/monitoring-setup) and [Security Practices](/team/devops/security-practices).
