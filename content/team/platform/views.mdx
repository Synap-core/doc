# Views

Views are named queries plus rendering configuration over shared entities.

- 16 view types are defined, with 12 currently implemented.
- Same entity set can be rendered as table, kanban, calendar, flow, bento, and more.
- View config remains lightweight so data semantics stay in profiles/entities.

See also: [Platform index](/team/platform) and [Synap packages](/team/synap-packages).
