# Relay

Target Users: Mobile engineers, product/design teammates working on mobile UX, operational teammates coordinating launch readiness
Marketing: The mobile surface of Synap — capture on the go, feed consumption, and native OS integrations that make your knowledge work feel effortless
Key User Stories:
- "I want to capture anything from my phone — a link, a text, a photo — and have it structured in my pod"
- "I want a morning briefing delivered through my phone's feed with smart organization"
- "I need native share, spotlight search, and widgets that connect my phone to Synap seamlessly"
- "I want to switch between my personal pod and team workspaces instantly from my phone"

## Overview

Relay is Synap's **mobile-first surface** built on Expo/React Native. It's not just a responsive version of the desktop app — it's a deliberately mobile-optimized experience that takes advantage of native device capabilities: camera, location, push notifications, share extensions, widgets, and OS search indexing.

Relay serves three primary surfaces:
1. **Feed consumption** — morning briefing, signal feed, proactive insights delivered as a scrollable experience optimized for mobile
2. **Capture** — quick entity creation from the camera, clipboard, share sheet, or voice
3. **Native OS integrations** — share extensions, widgets, live activities, spotlight search, and shortcuts that make Synap feel like a native part of the device

Under the hood, Relay delegates complex native integrations to 7 native OS sub-packages: share, spotlight, widgets, live activities, intents, shared storage, and on-device vision.

## Architecture

```
┌─────────────────────────────────────────────────────┐
│                   React Native (Expo)               │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌────┐ │
│  │ Feed     │  │ Capture  │  │ Entities │  │ ...│ │
│  │ Surface  │  │ Surface  │  │ Surface  │  │    │ │
│  └────┬─────┘  └────┬─────┘  └────┬─────┘  │    │ │
└───────┼──────────────┼──────────────┼────────┼────┘
        │              │              │
┌───────┼──────────────────────────────────────────────┐
│  Native OS Layer (native-os-* packages)              │
│  share · spotlight · widgets · intents · live-       │
│  activities · shared-storage · vision                │
└──────────────────────────────────────────────────────┘
```

**Tech Stack:** Expo (React Native) · Tamagui (cross-platform UI) · Zustand · tRPC
**Repo:** `relay-app/` (Expo standalone app)
**Native OS:** `synap-app/packages/core/native-os-*` (7 sub-packages)
**Key files:** `relay-app/app/` (Expo file-based routing), `relay-app/src/` (components)
