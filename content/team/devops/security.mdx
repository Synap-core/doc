# Security

Security posture combines identity hardening, infra controls, and auditability.

- ES256 JWT verification with JWKS rotation support.
- Strict secret handling for service-to-service credentials.
- Host-level network controls and documented operational guardrails.
- Auditable logs for critical auth and mutation paths.

See also: [Security practices](/team/devops/security-practices).
