### Phase 2: Build Scene 1 (2-3 hours)

**Test your component library with the opening scene:**

```bash
npm start
# Opens http://localhost:3000
```

**Create:** `src/compositions/DeathOfSaaS/Scene01-Opening.tsx`

```tsx
import React from "react";
import { AbsoluteFill } from "remotion";
import { GradientText } from "../../components/text/GradientText";
import { ParticleField } from "../../components/effects/ParticleField";
import { Vignette } from "../../components/effects/Vignette";
import { FilmGrain } from "../../components/effects/FilmGrain";

export const Scene01Opening: React.FC = () => {
  return (
    <AbsoluteFill style={{ backgroundColor: "#0A0A0A" }}>
      {/* Background layers */}
      <ParticleField count={150} speed={0.3} direction="up" opacity={0.2} />
      <Vignette intensity={0.4} />
      <FilmGrain intensity={0.04} />

      {/* Main title */}
      <AbsoluteFill style={{ justifyContent: "center", alignItems: "center" }}>
        <GradientText
          from="#38BDF8"
          to="#F8FAFC"
          fontSize={120}
          fontWeight={900}
          animate="scaleIn"
          delay={10}
        >
          The Death of SaaS
        </GradientText>
      </AbsoluteFill>
    </AbsoluteFill>
  );
};
```

**Test in Remotion Studio:**
1. Scrub timeline - does animation feel smooth?
2. Adjust `delay` and `fontSize` to taste
3. Try different `ParticleField` counts

### Phase 4: Add Transitions (2-3 hours)

Use `@remotion/transitions` for smooth scene changes:

```tsx
import { TransitionSeries, linearTiming } from "@remotion/transitions";
import { fade } from "@remotion/transitions/fade";
import { slide } from "@remotion/transitions/slide";

<TransitionSeries>
  <TransitionSeries.Sequence durationInFrames={300}>
    <Scene01Opening />
  </TransitionSeries.Sequence>

  <TransitionSeries.Transition
    presentation={fade()}
    timing={linearTiming({ durationInFrames: 15 })}
  />

  <TransitionSeries.Sequence durationInFrames={600}>
    <Scene02ScatteredTools />
  </TransitionSeries.Sequence>

  {/* ... more scenes ... */}
</TransitionSeries>
```

**Transition Guide:**
- Fast cuts: 10 frames
- Normal: 15-20 frames
- Dramatic: 30 frames
- Fade for mood shifts
- Slide for narrative progression

### Phase 5: Audio Integration (2-3 hours)

**Add background music:**

```tsx
import { Audio } from "remotion";

<Audio
  src={staticFile("audio/music/death-of-saas.mp3")}
  volume={0.3}
  startFrom={0}
/>
```

**Add voiceover:**

```tsx
<Audio
  src={staticFile("audio/voiceover/death-of-saas.mp3")}
  volume={1}
  startFrom={0}
/>
```

**Add sound effects:**

```tsx
import { Audio, useCurrentFrame } from "remotion";

// Glass shatter at frame 300 (10 seconds)
{frame >= 300 && (
  <Audio
    src={staticFile("audio/sfx/glass-shatter.wav")}
    volume={0.8}
    startFrom={300}
  />
)}
```

**Sync timing:**
1. Play video with voiceover
2. Note exact moments for SFX
3. Add SFX at those frames
4. Adjust music volume during voiceover: `volume={0.2}`

## 🚧 Troubleshooting

### "Out of disk space"
```bash
# Clear caches
npm cache clean --force
rm -rf node_modules/.cache
rm -rf .remotion

# Check space
df -h
```

### "Rendering is slow"
- Reduce particle counts (100-200 max)
- Lower quality temporarily: `--quality 80`
- Render smaller first: `--scale 0.5`
- Use `--every-nth-frame 2` for draft

### "Audio not syncing"
- Use `<Audio startFrom={frameNumber}>` not `delay`
- Check FPS is consistent (30fps everywhere)
- Render with `--enforce-audio-track` flag
