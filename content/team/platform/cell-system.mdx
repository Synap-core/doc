# Cell System

Cells are the universal render unit across all Synap surfaces.

- 80+ static registrations ship today across data, AI, governance, and communication categories.
- Hosts like bento, panel, and page containers render the same cell contracts.
- AI can compose existing cells and progressively introduce new capability-gated cell types.
