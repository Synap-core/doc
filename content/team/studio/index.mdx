## Overview

Studio is Synap's **single focused web app** — replacing the earlier multi-app approach (data, views, documents, channels as separate surfaces) with one coherent shell built on HeroUI. It's organized around three core concepts:

**Channels** — every agent + every object has a canonical conversation
**Documents** — rich markdown editing with AI proposal diffs
**Agents** — your AI team, summoned via @mention or called directly

Plus supporting surfaces: Tasks, Inbox, Graph, Entity detail, Home.

Unlike Browser, Studio **does not expose multi-workspace to the user**. Every Studio session is one implicit workspace — no workspace switcher, no "which workspace am I in", no per-workspace settings. The model: free tier = shared pod (one workspace), Pro tier = own dedicated pod (one workspace). Upgrading means "get your own pod."

## Architecture

| Dimension | Value |
|-----------|-------|
| Location | `synap-app/apps/studio/` |
| Port | 3002 |
| Framework | Next.js 15 (App Router) |
| UI | React 19 · HeroUI v2 · Tailwind v3 · next-themes |
| Data | tRPC · Socket.IO (realtime) · Zustand |
| Provider chain | 10 providers · 2 Zustand stores · 3 mapper files |
| Editor | Tiptap (collaborative markdown) |
| Graph | reagraph |
| Aesthetic | "ChatGPT 2028" — premium, calm, no shadows, semantic tokens, Fraunces headings |
