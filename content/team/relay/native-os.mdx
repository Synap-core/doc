# Native OS

Relay uses the `@synap-core/native-os` bridge to expose device capabilities safely.

- Umbrella package coordinates multiple platform-specific modules.
- Maintains Metro-safe static requires and explicit capability boundaries.
- Keeps app features portable across iOS/Android while preserving native hooks.

See also: [Native OS UX](/team/relay/native-os-ux).
