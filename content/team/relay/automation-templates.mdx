## Flow 9: Daily Usage Pattern

**Typical morning routine:**

```
1. Open app → Feed tab
2. Scan highlights (AI-pinned items at top)
3. See: "3 new messages processed overnight"
   → Tap → see AI summaries of Telegram conversations
4. See: "ETHGlobal Cannes in 2 days"
   → Tap → review prep briefing
5. See reconnection nudge for Marie
   → Tap Message → send quick Telegram DM
6. Close app (< 3 minutes total)
```

**Post-event capture:**

```bash

1. At event: tap FAB, voice capture 3 new contacts
2. Evening: open Person Detail for each
   → Review AI extractions, add manual notes
   → Tag all with event name
3. Next day: feed shows "3 new contacts from ETHGlobal"
   → AI already extracted skills and topics from names + context

```
