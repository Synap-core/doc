# Professional Component Library Guide

This is your arsenal of reusable, production-ready components for creating stunning motion graphics in Remotion.

---

## 📝 Text Components

### 1. GradientText
Beautiful gradient text with animations.

```tsx
import { GradientText } from "./components/text/GradientText";

<GradientText
  from="#38BDF8"
  to="#F8FAFC"
  angle={135}
  fontSize={96}
  fontWeight={800}
  animate="scaleIn"
  delay={30}
>
  The Death of SaaS
</GradientText>
```

**Props:**
- `from`, `to`: Gradient colors
- `angle`: Gradient angle (degrees)
- `fontSize`: Font size in pixels
- `fontWeight`: 400-900
- `animate`: `"fadeIn"`, `"scaleIn"`, or `"none"`
- `delay`: Start animation after N frames

---

### 2. TypewriterText
Classic typewriter effect with optional cursor.

```tsx
import { TypewriterText } from "./components/text/TypewriterText";

<TypewriterText
  startFrame={60}
  charsPerFrame={2}
  cursor={true}
  cursorBlinkSpeed={16}
  style={{ fontSize: 48, color: "#F8FAFC" }}
>
  Your data. Your rules. Any interface you want.
</TypewriterText>
```

**Props:**
- `startFrame`: When to start typing
- `charsPerFrame`: Speed (lower = faster)
- `cursor`: Show blinking cursor
- `cursorBlinkSpeed`: Frames per blink cycle

---

### 3. HighlightText


## ✨ Visual Effects

### 5. ParticleField
Floating particles with customizable behavior.

```tsx
import { ParticleField } from "./components/effects/ParticleField";

<ParticleField
  count={200}
  speed={0.5}
  color="#38BDF8"
  opacity={0.3}
  size={[2, 6]}
  direction="up"
/>
```

**Directions:**
- `"up"`: Rising particles (dust in light)
- `"down"`: Falling particles (rain, snow)
- `"random"`: Organic drift
- `"left"`, `"right"`: Horizontal flow

**Performance:** Keep count under 300 for smooth rendering.

---

### 6. Vignette
Subtle edge darkening for cinematic focus.

```tsx
import { Vignette } from "./components/effects/Vignette";

<Vignette intensity={0.5} color="#000000" />
```

**Best practices:**
- Use `0.3-0.5` intensity for subtlety
- Add to dramatic scenes, testimonials, closeups

---

### 7. FilmGrain
Organic film texture overlay.

```tsx
import { FilmGrain } from "./components/effects/FilmGrain";

<FilmGrain intensity={0.05} />
```

**Tip:** Lower is better. `0.03-0.08` range looks natural.

---

### 8. ChromaticAberration
RGB channel separation effect.

```tsx

<ChromaticAberration intensity={5} startFrame={0} endFrame={30}>
  <YourContent />
</ChromaticAberration>
```

**Use for:** Glitch moments, VHS aesthetic, impact frames.

## 🎬 Real-World Composition Examples

### Example 1: Opening Title with Multiple Effects

```tsx
import { AbsoluteFill } from "remotion";
import { GradientText } from "./components/text/GradientText";
import { ParticleField } from "./components/effects/ParticleField";
import { Vignette } from "./components/effects/Vignette";
import { FilmGrain } from "./components/effects/FilmGrain";

export const OpeningScene = () => {
  return (
    <AbsoluteFill style={{ backgroundColor: "#0A0A0A" }}>
      {/* Background effects */}
      <ParticleField count={150} speed={0.3} direction="up" opacity={0.2} />
      <Vignette intensity={0.4} />
      <FilmGrain intensity={0.04} />

      {/* Main title */}
      <AbsoluteFill style={{ justifyContent: "center", alignItems: "center" }}>
        <GradientText
          from="#38BDF8"
          to="#F8FAFC"
          fontSize={120}
          fontWeight={900}
          animate="scaleIn"
          delay={10}
        >
          The Death of SaaS
        </GradientText>
      </AbsoluteFill>
    </AbsoluteFill>
  );
};
```

### Example 3: Dramatic Glitch Transition

```tsx

                BROKEN

      {/* Next scene */}
      <Sequence from={90}>

    </>

};
```

## 🎯 Advanced Techniques

### Combining Multiple Text Effects

```tsx
<GradientText from="#38BDF8" to="#F8FAFC" fontSize={72} animate="fadeIn">
  <TypewriterText charsPerFrame={3}>
    The future is{" "}
  </TypewriterText>
  <HighlightText highlightColor="#10B981" startFrame={60}>
    open source
  </HighlightText>
</GradientText>
```

### Stacking Effects for Depth

```tsx
<AbsoluteFill>
  {/* Layer 1: Background */}
  <ParticleField count={200} opacity={0.1} />

  {/* Layer 2: Main content */}
  <YourContent />

  {/* Layer 3: Foreground effects */}
  <FilmGrain intensity={0.05} />
  <Vignette intensity={0.5} />
</AbsoluteFill>
```

### Synchronized Animations

```tsx
const ENTRANCE_DELAY = 30;

<>
  <GradientText animate="scaleIn" delay={ENTRANCE_DELAY}>
    Title
  </GradientText>

  <HighlightText startFrame={ENTRANCE_DELAY + 60}>
    Highlighted word
  </HighlightText>

  <DataPod3D animateIn startFrame={ENTRANCE_DELAY} />
</>
```

## 🚀 Performance Tips

### Do's ✅
- Keep particle counts under 300
- Use `spring()` with `damping: 200` for smooth animations
- Premount sequences: `<Sequence premountFor={30}>`
- Memoize heavy computations with `React.useMemo()`

### Don'ts ❌
- Don't use CSS animations (use `interpolate()` instead)
- Don't create particles in render loop (use `useMemo`)
- Don't exceed 5000 particles total
- Don't use `useState` for animations (use `useCurrentFrame`)
