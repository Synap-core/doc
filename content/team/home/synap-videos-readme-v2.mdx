## 🚀 Quick Start

### 1. Launch Remotion Studio

```bash
cd synap-videos
npm start
```

### 2. Select a Composition

In Remotion Studio, choose:
- **DeathOfSaaS-V2** (Featured - New professional version)
- DeathOfSaaS (Original version)
- 3DShowcase (3D component demos)
- EventChainExplained (Event sourcing explanation)

### 3. Preview and Iterate

- Scrub the timeline
- Play in real-time
- Adjust timing
- Tweak colors

### 4. Render

```bash
# Full quality
npm run build -- --codec=h264 --composition=DeathOfSaaS-V2

# Draft (faster)
npm run build -- --codec=h264 --composition=DeathOfSaaS-V2 --scale=0.5
```

## 🎨 Design Philosophy

### Isometric Entity Blocks

Inspired by LEGO, each data type is a beautiful 3D block:
- Geometric (structured data)
- Gradients (visual depth)
- Icons (instant recognition)
- Studs (stackable/connectable)

### Connection Lines

Neural synapse-inspired connections:
- **Dependency** (Green) - Solid directional
- **Assignment** (Blue) - Pulsing
- **Hierarchy** (Purple) - Thick flow
- **Related** (Amber) - Dotted

### Modern Motion Design

- Spring physics for natural movement
- Staggered entrances for flow
- Particle systems for energy
- Gradient text for polish

## 🔧 Technical Details

### Built With
- **Remotion 4.0** - Video framework
- **React 18** - UI library
- **TypeScript** - Type safety
- **Three.js** - 3D graphics (optional)

### Requirements
- Node.js 18+
- 8GB RAM minimum
- Modern GPU recommended for smooth preview

### Performance Tips
1. Use `scale=0.5` for draft renders
2. Limit particle counts to 300 max
3. Preview at 720p, render at 1080p
4. Use `concurrency` flag for faster renders

## 🎯 Creating New Videos

### Option 1: Use Existing Scenes

Mix and match scenes from "Death of SaaS":

```tsx
import { Series } from "remotion";
import { Scene02ScatteredTools } from "./DeathOfSaaS-v2/Scene02-ScatteredTools";
import { Scene05NewArchitecture } from "./DeathOfSaaS-v2/Scene05-NewArchitecture";

export const MyVideo = () => (
  <Series>
    <Series.Sequence durationInFrames={600}>
      <Scene02ScatteredTools />
    </Series.Sequence>
    <Series.Sequence durationInFrames={750}>
      <Scene05NewArchitecture />
    </Series.Sequence>
  </Series>
);
```

### Option 2: Build Custom Scenes

```tsx
import { AbsoluteFill } from "remotion";
import { EntityBlock, GradientText, ParticleField } from "../components";

export const CustomScene = () => (
  <AbsoluteFill style={{ backgroundColor: "#020617" }}>
    <ParticleField count={150} color="#38BDF8" />

    <div style={{ position: "absolute", top: "50%", left: "50%", transform: "translate(-50%, -50%)" }}>
      <GradientText from="#38BDF8" to="#10B981" fontSize={72}>
        Your Message
      </GradientText>
    </div>

    <div style={{ position: "absolute", bottom: 100, left: 100 }}>
      <EntityBlock type="project" size={100} animateIn />
    </div>
  </AbsoluteFill>
);
```

## 💡 Tips for Success

**Visual Design:**
- Less is more - don't overcomplicate
- Use the color palette consistently
- Let animations breathe (don't rush)
- Add subtle particles for depth

**Animation Timing:**
- Entrance: 30-60 frames (1-2 seconds)
- Hold: 60-90 frames (2-3 seconds)
- Exit: 20-30 frames (0.5-1 second)

**Performance:**
- Test on target hardware
- Render overnight if needed
- Use background rendering
- Preview at lower quality
