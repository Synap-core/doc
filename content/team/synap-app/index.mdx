`synap-app` is mostly a frontend package workspace and integration layer.

## Purpose

- Track current scope vs deprecated scope in this repo.
- Clarify package ownership and layering decisions.
- Document frontend integration paths with backend and intelligence services.

## Target users

- Frontend and full-stack engineers
- Platform contributors touching shared packages
- Product/design teammates needing package-level orientation

- The **Telegram app in this repo is deprecated** and not the forward product direction.
