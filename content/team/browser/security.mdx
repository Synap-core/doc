## Core posture

- sandbox external web content
- block dangerous URL schemes
- keep secrets in OS secure storage
- avoid token exposure in renderer and logs

## Release gates

- IPC allowlist and schema validation reviewed
- webview sandbox posture re-verified
- updater integrity and publish target verified
- pod URL and transport constraints validated
