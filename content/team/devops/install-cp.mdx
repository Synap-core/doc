# Install Control Plane

Control Plane install bootstraps the management backbone for pods and billing.

- Use the scripted setup flow (`./synap-cp install`) for deterministic provisioning.
- Generate and store secrets before exposing public endpoints.
- Start services, then bootstrap initial admin and verify health endpoints.

Continue with [Security Practices](/team/devops/security-practices) and [Incident Runbook](/team/control-plane/incident-runbook).
