## 🎨 Customization Tips

### Change Colors

Edit `src/components/` files and search for:
- `#38BDF8` → Your primary color
- `#0A0A0A` → Background color
- `#F8FAFC` → Text color

### Adjust Timing

In each `Scene0X-*.tsx` file, change:
```typescript
const opacity = spring({
  frame: frame - 60,  // ← Change this delay
  fps,
  config: { damping: 200 },  // ← Higher = faster
});
```

### Modify Text

All text is in the scene files - just edit the strings!

## 📹 Rendering Final Video

### Quick Preview Render (Draft)

```bash
npx remotion render DeathOfSaaS out/preview.mp4 --quality 80
```

**Time:** ~2-3 minutes
**Size:** ~50MB
**Use for:** Quick review, sharing for feedback

---

### High Quality Render (Final)

```bash
npx remotion render DeathOfSaaS out/death-of-saas-final.mp4 --quality 95
```

**Time:** ~5-8 minutes
**Size:** ~150MB
**Use for:** Final export, publishing

---

### 4K Render (Premium)

```bash
npx remotion render DeathOfSaaS out/death-of-saas-4k.mp4 \
  --width 3840 \
  --height 2160 \
  --quality 100
```

**Time:** ~15-20 minutes
**Size:** ~500MB
**Use for:** High-end displays, future-proofing

---

### Different Formats

```bash
# WebM (web-friendly)
npx remotion render DeathOfSaaS out/video.webm

# ProRes (for editing in Premiere/Final Cut)
npx remotion render DeathOfSaaS out/video.mov --codec prores

# MP4 with specific bitrate
npx remotion render DeathOfSaaS out/video.mp4 --video-bitrate 8M
```

## 🐛 Troubleshooting

### "Remotion Studio won't start"

```bash
rm -rf .remotion node_modules/.cache
npm start
```

### "Particles are laggy"

Lower particle counts in scene files:
```typescript
<ParticleField count={100} />  // Instead of 200
```

### "Video renders slowly"

Use draft settings:
```bash
npx remotion render DeathOfSaaS out/video.mp4 \
  --quality 80 \
  --scale 0.5
```

### "TypeScript errors"

```bash
npx tsc --noEmit
```

This will show exact errors and line numbers.
