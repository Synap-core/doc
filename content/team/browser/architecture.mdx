## Process model

```text
Main process (Node)      - windows, secure storage, updater, IPC router
Renderer (React/Tamagui) - shell UI and product apps
Webviews (untrusted)     - external websites in sandboxed contexts
```

## Boundary rules

- keep `contextIsolation` enabled
- avoid Node APIs in untrusted webviews
- allowlist IPC channels and validate payloads
- prefer sensitive pod calls through trusted paths
