# Monitoring Setup

Monitoring combines platform health, pod metrics, and intelligence service visibility.

- Prometheus scrapes core service metrics.
- Grafana dashboards provide operations views and alerting surfaces.
- Intelligence Service metrics require explicit token wiring for secure access.

Pair setup with the incident runbook so on-call has actionable dashboards.
