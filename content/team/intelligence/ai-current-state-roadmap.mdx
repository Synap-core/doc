## Part 5: Risk Assessment & Mitigation

```
Risk: OpenAI/Anthropic increase prices 10x
Impact: Unit economics break (€2 → €20/user)
Likelihood: Low-Medium

Mitigation:
- Multi-model routing (use cheaper alternatives)
- Self-hosted open models (Llama, Mistral)
- User pays for premium models
- Model caching and optimization
```

```
Risk: System slow with 10K+ concurrent users
Impact: Bad UX, churn
Likelihood: Medium

Mitigation:
- Load testing before launch
- Auto-scaling infrastructure
- Caching strategies
- Regional deployments
- Performance monitoring
```
