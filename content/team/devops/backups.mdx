# Backups

Backup operations protect both structured data and deployment state.

- Manual DB snapshots for emergency and migration workflows.
- Pod archive flow for tenant transfer and recovery scenarios.
- S3-compatible storage configuration for off-host retention.

Operationally pair this with restore drills in your incident runbook.
