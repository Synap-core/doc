# Browser

Target Users: Desktop engineers, security/release operators, product/design teammates working on the desktop experience
Marketing: The Electron desktop app — your sovereign Synap workspace, locally runnable with enterprise-grade security
Key User Stories: 
- "I want to install Synap on my laptop and start capturing knowledge immediately"
- "I need to capture web content seamlessly while browsing and have it appear as structured entities in my pod"
- "I want a trusted, sandboxed desktop experience where my data stays on my machine"
- "I need a reliable update path that ships without breaking my local setup"
Key Features:
| Feature | Description | Status | Repo |
|---------|-------------|--------|------|
| Desktop shell | Electron-based app with sidebar, 18+ embedded apps | Live | `browser/` |
| Web capture | Capture flow from web → structured entities | Live | `browser/` |

Current Status: **Live** — primary desktop product surface

## Overview

Browser is Synap's **primary user-facing product** — a desktop application built on Electron that provides a secure, offline-capable interface to your data pod. Unlike the web-only Studio app, Browser runs locally, giving users direct control over their data and infrastructure.

The app exposes 18 embedded apps (dashboard, browser, chat, library, data, views, calendar, files, channels, proposals, workflows, marketplace, relay, vault, settings, terminal, whiteboard, dataStructure) controlled by a single manifest file. Each app is a package in `synap-app/packages/apps/` or `synap-app/packages/features/` that gets lazily loaded.

Browser also serves as the primary surface for web capture — the entry point for knowledge work. Users can capture content from any website and have it transformed into structured entities in their pod.

## Architecture

```
┌─────────────────────────────────────────┐
│        Electron Main Process            │
│  ┌──────────┐  ┌──────────┐  ┌────────┐ │
│  │ Sidebar  │  │ IPC Hub  │  │ Updater│ │
│  └────┬─────┘  └────┬─────┘  └────────┘ │
└───────┼──────────────┼──────────────────┘
        │              │
┌───────┼──────────────────────────────────┐
│       ▼     Renderer (React + Tamagui)   │
│  ┌──────────┐  ┌──────────┐  ┌────────┐ │
│  │ App Shell│  │ Cell     │  │ Capture│ │
│  │(18 apps) │  │ Runtime  │  │ Flow   │ │
│  └──────────┘  └──────────┘  └────────┘ │
└─────────────────────────────────────────┘
```

**Tech Stack:** Electron · React 19 · Tamagui · Zustand · tRPC · Yjs (collaborative editing)
**Repo:** `browser/` (standalone monorepo package)
**Entry Point:** `electron/main.ts` (main process), `electron/renderer/src/` (renderer)
**Sidebar Config:** `electron/renderer/src/apps/manifest.ts`
