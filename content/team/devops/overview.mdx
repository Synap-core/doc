# DevOps Overview

DevOps operations center on two managed control surfaces:

- **Control Plane** (`api.synap.live`) for provisioning, billing, and pod lifecycle.
- **Intelligence Service** (`ai.synap.live`) for agent execution and orchestration.

Both are operated via scripted install/update flows, with explicit health checks and incident playbooks.
