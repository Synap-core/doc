# Quick 3D Reference Card

## 🚀 Basic 3D Scene Template

```tsx
import React from "react";
import { AbsoluteFill, useCurrentFrame, useVideoConfig } from "remotion";
import { ThreeCanvas } from "@remotion/three";

export const My3DScene: React.FC = () => {
  const frame = useCurrentFrame();
  const { width, height } = useVideoConfig();

  const rotation = frame * 0.02;

  return (
    <AbsoluteFill style={{ backgroundColor: "#0A0A0A" }}>
      <ThreeCanvas width={width} height={height} camera={{ position: [0, 0, 5], fov: 45 }}>
        {/* Lighting */}
        <ambientLight intensity={0.4} />
        <directionalLight position={[5, 5, 5]} intensity={0.8} />

        {/* Your mesh */}
        <mesh rotation={[0, rotation, 0]}>
          <boxGeometry args={[2, 2, 2]} />
          <meshStandardMaterial color="#38BDF8" />
        </mesh>
      </ThreeCanvas>
    </AbsoluteFill>
  );
};
```

## 💡 Common Lighting Setups

### Basic (Good for most scenes)
```tsx
<ambientLight intensity={0.4} />
<directionalLight position={[5, 5, 5]} intensity={0.8} />
```

### Three-Point Lighting
```tsx
<ambientLight intensity={0.2} />
<directionalLight position={[5, 5, 5]} intensity={0.8} /> {/* Key */}
<directionalLight position={[-5, 0, -5]} intensity={0.3} /> {/* Fill */}
<directionalLight position={[0, -5, 0]} intensity={0.2} /> {/* Back */}
```

### Point Light (Glow)
```tsx
<pointLight
  position={[0, 0, 0]}
  intensity={2}
  color="#38BDF8"
  distance={5}
  decay={2}
/>
```

## 🎬 Animation Patterns

### Rotation
```tsx
const frame = useCurrentFrame();
const rotation = frame * 0.02;

<mesh rotation={[0, rotation, 0]} />
```

### Float/Bob
```tsx
const y = Math.sin(frame * 0.05) * 2;

<mesh position={[0, y, 0]} />
```

### Orbit
```tsx
const angle = frame * 0.01;
const radius = 3;
const x = Math.cos(angle) * radius;
const z = Math.sin(angle) * radius;

<mesh position={[x, 0, z]} />
```

### Pulse/Scale
```tsx
const scale = 1 + Math.sin(frame * 0.1) * 0.2;

<mesh scale={[scale, scale, scale]} />
```

### Spring Entrance
```tsx
const entrance = spring({
  frame: frame - startFrame,
  fps,
  config: { damping: 100 },
});

const scale = interpolate(entrance, [0, 1], [0, 1]);
```

## 🎯 Camera Positions

```tsx
{/* Angled view */}

{/* Top-down */}
camera={{ position: [0, 5, 0] }}

{/* Animated camera */}

const x = Math.sin(frame * 0.01) * 3;
camera={{ position: [x, 0, 5] }}
```

## 🐛 Troubleshooting Checklist

- [ ] Using `<ThreeCanvas>` not `<Canvas>`?
- [ ] Passed `width` and `height` props?
- [ ] Using `useCurrentFrame()` not `useFrame()`?
- [ ] All animations based on `frame`, not `delta`?
- [ ] Added lighting (`ambientLight` + `directionalLight`)?
- [ ] No `OrbitControls` or interactive controls?
- [ ] No `useState` for animation state?
- [ ] Using `random("seed")` not `Math.random()`?
