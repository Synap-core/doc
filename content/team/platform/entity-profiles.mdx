# Entity Profiles

Profiles define entity shape, behavior, and scope.

- Synap ships with 14 system profiles as default building blocks.
- `entityScope` controls whether entities are pod-wide or workspace-scoped.
- Custom profiles let teams add domain schemas without backend code changes.
- Story-based onboarding can scaffold profiles + views from presets.
