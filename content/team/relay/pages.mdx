### Feed Card Types

**Conversation card**
- Avatar + person name + timestamp
- AI summary of conversation (1-2 lines)
- Source badge (Telegram/LinkedIn/Phone/Manual)
- Topic pills
- Tap вҶ’ Person Detail, focused on this interaction

**Social signal card**
- Avatar + person name
- Activity description ("posted about X on LinkedIn")
- Preview snippet if available
- Tap вҶ’ Person Detail

**New encounter card**
- "New person detected" badge
- Name/handle + where detected (group name)
- Message count + timespan
- Actions: [Add to contacts] [Ignore]

**Reconnection nudge card**
- Person avatar + "You haven't talked to X in Y months"
- Reason: "They just posted about needing a CTO"
- Actions: [Message] [Remind later] [Dismiss]

**Cluster alert card**
- Topic/theme icon
- "N people in your network are working on [topic]"
- Person avatars row
- Tap вҶ’ filtered People view

**Event context card**
- Calendar icon + event name + days until
- "N people in your network attending"
- Person avatars row
- Actions: [View prep briefing] [See attendees]

### States
- **Empty:** First-time user, no data yet вҶ’ "Connect Telegram to start building your network" CTA
- **Loading:** Skeleton cards
- **Populated:** Scrolling feed
- **Filtered:** Active filter chips shown above feed

### Filters
- Chip bar below header: All | Conversations | Signals | Encounters | Nudges | Events
- Long press chip вҶ’ sub-filter (by person, source, topic)

### Person Card (Grid)
- Photo (circle, 64px)
- Name (bold, 1 line)
- Role @ Organization (muted, 1 line)
- Top 2-3 tag pills (skills/topics)
- Last interaction: relative time + source icon
- Strength indicator: subtle color ring around photo (green=strong, amber=fading, gray=cold)

### Layout вҖ” List Mode
- Compact rows: [Photo 32px] [Name] [Role] [Tags] [Last seen] [Strength dot]
- Sortable columns: Name, Last interaction, Strength, Organization

### Layout вҖ” Graph Mode
- Full-screen force-directed graph
- Nodes: person photos (circle), sized by interaction count
- Edges: thickness = interaction frequency, opacity = recency
- Clusters auto-detected by community
- Tap node вҶ’ Person Detail in bottom sheet (mobile) or side panel (desktop)
- Pinch to zoom, pan to navigate
- Legend: node size = interaction count, edge thickness = frequency
- Filter panel overlays on top

### Filter Panel
- **Source:** Telegram, LinkedIn, Phone, Manual, All
- **Skills/Tags:** Multi-select from extracted tags
- **Events:** "Met at [event]"
- **Recency:** Last 7d / 30d / 90d / All
- **Organization:** Company filter
- **Strength:** Strong / Warm / Cold
- **AI Query:** Free text вҶ’ "Who knows Solidity in Paris?"

### States
- **Empty:** "No contacts yet" вҶ’ Connect Telegram CTA
- **Searching:** Results update as you type
- **Filtered:** Active filter count badge, clear all button
- **Graph loading:** Progressive node rendering (show 20, then expand)

### Identity Section
- Photo: largest available across sources (Telegram > LinkedIn > generated avatar)
- Name: merged best quality
- Identity links: tappable icons for each connected channel (opens in native app)
- AI one-liner: auto-generated, editable
- Tags: auto-extracted + manually added, tappable for filter

### Timeline Tab
- Chronological interactions, grouped by month
- Each entry: source icon, date, AI summary
- Expandable: shows full conversation or raw data
- Entry types: Telegram message session, LinkedIn interaction, call, event attendance, manual note
- AI period summaries between months: "In Q1 2026, you discussed X and Y across N conversations"

### Intelligence Tab
- **Current focus areas** вҖ” AI-inferred from recent activity + conversations
- **Shared interests** вҖ” Overlap between their topics and yours
- **You could help** / **They could help** вҖ” AI suggestions based on graph
- **Mutual connections** вҖ” People you both know (from graph)
- **Reconnection reason** вҖ” If applicable, why to reach out now
- **Strength meter** вҖ” Visual indicator with breakdown (frequency, recency, depth)

### Notes Tab
- Manual notes, freeform
- Each note timestamped
- Rich text (markdown)
- AI can suggest notes from conversations ("add follow-up: discuss compliance at next call?")

### Actions Menu (В·В·В·)
- Edit person
- Merge with another person
- View in graph (switches to People graph view, centered on this person)
- Share contact
- Archive / Delete

   вҶ’ Immediate: create Person entities, merge with Telegram matches

### Key Moments
- After Telegram connect: show counter "Found 142 contacts, processing..."
- After contacts merge: "Matched 37 phone contacts with Telegram users"
- After first capture: "Created 3 new contacts from your note"
- Feed populates progressively during onboarding вҖ” user sees value immediately
