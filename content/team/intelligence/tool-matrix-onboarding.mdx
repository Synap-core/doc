# IS tools — onboarding & feed alignment

**Purpose:** Show which **tools** the orchestrator should call by **primary goal** (`@synap-core/onboarding-core`). Keep in sync with `synap-intelligence-service/.../tools/tool-registry.ts`.

| Primary goal (app default) | Primary tools | Notes |
|---------------------------|---------------|--------|
| `feed` (Relay) | `classify_feed_items`, `recommend_feed_sources`, `get_feed_preferences` / `update_feed_preferences`, feed trigger via pod | RSS transport is pod; IS classifies |
| `home_bento` (Browser) | Widgets / workspace proposal tools (when wired), `get_feed_preferences` optional | Bento seeding may be pod tRPC |
| `project` (Content) | Entity / project creation via Hub + proposals | Governance for mutations |
| `import` | Import pipeline + entity tools | PII review |
| `template` | Workspace template install, pack proposals | Multi-agent OS templates |

**Prompts:** `agents/prompts/universal-onboarding/` — `composeUniversalOnboardingPrompt`.
