## Command categories (draft)

- quick capture
- quick lookup
- action triggers that escalate into governed proposals when needed

## User flow baseline

1. invoke command
2. resolve identity/pod context
3. run read or propose write action
4. return short confirmation with link to full surface if needed

## UX constraints

- keep first response low-friction
- avoid hidden side effects
- expose failure and fallback clearly
